# laravel-szamlazzhu

Implementation for Laravel which was made to support Szamlazz.hu Agent version of **3.4** and higher.

## Installation

```bash
composer require illionitgroup/laravel-szamlazzhu
```

## Requirements

Supports Laravel: **5.5** / **6.*** / **7.*** / **8.*** / **9.*** / **10.***
